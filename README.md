# fullpage

### 使用方法

* [下载](https://github.com/ranyunlong/fullpage/archive/master.zip/) 

```javascript
<link rel="stylesheet" href="dist/css/fullpage.css">
<script src="src/js/fullpage.js"></script>
<script type="text/javascript">
    fullpage({
        el: '.page', //绑定fullpage
        optio2n: '.section', //页面项目
    });
</script>
```
